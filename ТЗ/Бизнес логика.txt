ОБЩАЯ ИНФОРМАЦИЯ:

Язык программирования: C#
Графический интерфейс: WPF
Формат сохраняемых данных: JSON

ВКЛАДКИ:

1."Добавить парсинг"
	Поле для ввода URL сайта, с которого нужно собирать данные
	Настройки парсинга, включая регулярные выражения и другие параметры, необходимые для обработки страницы
	Поле для ввода названия парсинга
Кнопка "Добавить", которая сохраняет параметры парсинга
2."Логи по каждому парсингу"
	Список выполненных парсингов с возможностью выбора любого из них для просмотра логов
	Логи, содержащие подробную информацию о каждом парсинге, включая дату, время, URL, 
	количество собранных данных и другие параметры
3."Добавить прокси"
	Поле для ввода списка прокси в формате IP:Port
	Кнопка "Добавить", которая сохраняет прокси в базе данных и делает их доступными для использования в парсере
4."Управление настройками"
	Возможность изменять настройки парсера, такие как частота запросов, тайм-ауты, максимальное количество страниц 
	для парсинга и т.д.
	Возможность настройки формата сохраняемых данных (например, выбор между JSON и XML)
	Возможность добавления дополнительных настроек, таких как настройки прокси-серверов или параметры 
	авторизации на сайте, если это необходимо
5."Справка"
	Раздел с описанием функциональности парсера и инструкциями по его использованию
	Раздел с часто задаваемыми вопросами (FAQ) и ответами на них
6."О программе"
	Информация о версии программы, разработчиках и другие сведения о программе.
